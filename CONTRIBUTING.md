# CONTRIBUTING
Open Source DNS Sinkhole

We welcome contributions of all kinds. This is a community project.
Ways to help:
- Report bugs and issues
- Suggest features
- Submit code (Go or TypeScript)
- Improve documentation
- Test on different hardware
